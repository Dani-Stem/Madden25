# Madden 2k25
A fun Madden football inspired python game.

## Setup Instructions

1. Make the Script Executable on macOS/Linux:
    ```sh
    chmod +x setup.sh
    ```
2. Run the Script:
-  On macOS/Linux, run:
    ```sh
    ./setup.sh
    ```
- On Windows, you can run the script in a Git Bash or other bash-compatible shell.
